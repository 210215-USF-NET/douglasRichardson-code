Type of Selectors

Simple Selectors

  • Tag/Element
    • uses the tag name to identify the element
    • this will affect all elements with that tag name
  • Class
    • uses the class name to style a group of elements
    • denoted by .className
  • Id
    • Uses an unique id
    • denoted by #id
  • Pseudo-class
    • specific keyword at the end of the selector used to style the elements when it is at a certain state
    • :active :hover etc
  • Pseudo-element
    • used to style specific parts of an element
    • ::before ::after ::firstline ::firstletter

Combinator Selectors

  • Child descendant
  • Sibling
  • attribute selectors
  • Multiple selectors

Wildcard selector

  • The *
  • Styles everything, all elements
